### Technologies

### Description
Developed a comprehensive full-stack web application for a registered NGO in Rajasthan, India, enabling secure online donations, showcasing organizational initiatives, and providing administrative tools for donation management.

### Key Achievements

• Built a **full-stack donation platform** with React frontend and Node.js/Express backend, integrating Razorpay payment gateway for secure online transactions with cryptographic signature verification

• Developed a **comprehensive admin dashboard** featuring real-time donation statistics, advanced search/filter capabilities (by name, email, phone, payment ID, cause), pagination, and token-based authentication

• Implemented **bilingual support (English/Hindi)** using React Context API, managing 700+ translation strings across 15+ pages with dynamic language switching and persistent user preferences

• Designed and implemented **PostgreSQL database schema** with proper indexing, constraints, and migration scripts, optimizing queries for efficient data retrieval and reporting

• Created **RESTful API architecture** with 6 secure endpoints for payment processing, donation verification, and admin operations, including comprehensive error handling and validation

• Integrated **Razorpay payment gateway** with complete payment flow including order creation, signature verification, success/failure handling, and transaction logging to database

• Built **15+ responsive pages** and **20+ reusable components** including donation forms, admin panels, event management, image galleries, and team member profiles with smooth animations

• Deployed **production-ready application** on Vercel (frontend) and Render/Railway (backend), configuring environment variables, CI/CD pipelines, and optimizing build processes

• Implemented **security best practices** including payment signature verification, token-based authentication, CORS configuration, input validation, and SQL injection prevention

• Ensured **accessibility and performance** with semantic HTML, ARIA labels, mobile-first responsive design, code splitting, lazy loading, and optimized animations
